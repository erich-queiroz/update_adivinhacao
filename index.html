Adivinhe um número!

Overview

Este código é um estudo de caso para o desenvolvimento web.
Ele aborda conceitos importantes como manipulação do DOM (Document Object Model) e eventos JavaScript.
Além disso, também é possível ver a utilização de variáveis, funções, condicionais e lógica de programação. Este código é uma referência para a interação entre JavaScript e HTML.

Instruções

O objetivo do jogo é adivinhar um número entre 1 e 10 gerado aleatoriamente pelo código.
Você tem 10 tentativas para adivinhar o número correto.
Para fazer sua tentativa, insira o número que você acha que é o correto no campo de entrada.
Pressione a tecla "Enter" ou clique no botão "Tentar adivinhar" para enviar sua tentativa.
